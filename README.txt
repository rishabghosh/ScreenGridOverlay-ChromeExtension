# Screen Grid Overlay Extension

This Chrome extension automatically draws configurable vertical and horizontal lines over any website.

## ðŸš€ How to Install

1. Download and unzip this folder.
2. Open Chrome and go to `chrome://extensions/`.
3. Enable **Developer Mode** (top-right toggle).
4. Click **Load unpacked** and select the unzipped folder.
5. The extension should now appear in your toolbar.

## ðŸ§  How to Use

- Click the extension icon to open the popup.
- Use the form to add vertical or horizontal lines with:
  - Position in pixels
  - Color of the line
  - Thickness
- Click **Add Line**, then refresh the page to see the new lines.
- Click **Clear All** to remove all lines.

Lines appear on **every website**, across all tabs, after refresh.
